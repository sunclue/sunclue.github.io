Python 3

基础语法

  • 默认采用UTF-8编码

  • 标识符基本与c相同，但可采用中文作为变量名

  • 保留字，标准库中的keyword模块可以输出当前版本的所有关键字

  • 单行注释以#开头，多行注释可用'''和"""

  • 缩进代表代码块，无需{}，缩进的空格数可变，但同一代码块必须包含相同的缩进空格数

  • 可使用\实现多行语句，在[]，{}，()中的语句不需要\

  • 数字类型

    • int（整数）
    • bool（布尔）
    • float（浮点数）
    • complex（复数）

  • 字符串

    • 单引号与双引号使用完全相同
    • 使用三引号可指定一个多行字符串
    • 转义符\
    • 使用r可以让反斜杠不发生转义，r"hello world!\n"则\n不会转义
    • 连续的多个字符转换会转换为一整个字符串
    • +可连接字符串，*可重复字符串（后面跟数字）
    • 字符串索引方式，从左往右以0开始，从右往左以-1开始
    • 字符串不能改变
    • python无单独字符类型
    • 字符串截取语法格式：变量[头下标:尾下标:步长]，不包括尾下标上的字符，步长为负数时表示逆向

  • 空行

    • 函数之间或类的方法之间用空行分隔，表示一段新代码开始。类和函数入口之间也用空行分隔，以突出函数入口的开始

  • 同一行显示多条语句，语句之间使用;分隔

  • 多个语句构成代码组

    • 缩进相同的一组语句构成一个代码块
    • 像if，while，def和class这样的复合语句，首行以关键字开始，以冒号结束，该行之后的一行或多行代码构成代码组
    • 将首行及后面的代码组称为一个子句（clause）

  • print输出，默认换行，要实现不换行需要在变量末尾加上end=""

  • import与from…import，用于导入相应模块

    • 将整个模块导入，import somemodule
    • 从某个模块中导入某个函数，from somemodule import somefunction
    • 从某个模块导入多个函数，from somemodule import firstfunc,secondfunc
    • 将某个模块中的全部函数导入，from somemodule import *

基本数据类型

  • 变量不需要声明，在使用前需要赋值，赋值后变量才会被创建

  • 多个变量赋值

    • a=b=c=1
    • a,b,c=1,2,"hello"

  • 标准数据类型

    • number

      • 支持int、float、bool、complex
      • bool是int的子类，True和False可以和数字相加
      • 复数由a+bj或complex(a,b)表示，a，b都是浮点型

    • string

      • 不能改变，向一个索引位置赋值会引起错误

    • list

      • 可以完成大多数集合类的数据结构实现
      • 元素类型可以不相同，支持数字、字符串、列表
      • 列表是写在方括号[]之间，用逗号隔开的元素列表
      • 列表可以被索引和截取，列表被截取后返回一个包含所需元素的新列表，都与字符串类似
      • +是列表连接运算符，*是重复操作
      • 列表中元素可以改变

    • tuple（元组）

      • 与列表类似，不同之处在于元组的元素不能修改

      • 元组写在小括号()内，元素间用逗号隔开

      • 元素类型可以不相同

      • 同样可以索引、截取、连接、重复

      • 元组不可变，但可包含可变的元素

      • 额外的语法规则

      • string、list、tuple都属于sequence

    • set

      • 集合由一个或数个形态各异的大小整体构成的，构成集合的事物或对象称为元素或是成员
      • 基本功能是进行成员关系测试和删除重复元素
      • 可以使用大括号{}或set()函数创建集合，创建一个空集合只能用set(),{}是创建一个空字典
      • 集合中重复的元素会被自动去掉
      • 可以进行集合运算
        • -为差集
        • |为并集
        • &为交集
        • ^为两集合中不同时存在的元素

    • dictionary

      • 列表是有序的对象集合，字典是无序的对象集合，两者区别在于：字典当中的元素是通过键来存取的，而不是偏移

      • 字典是一种映射类型，用{}标识，是一个无序的键（key）：值（value）的集合

      • 键（key）必须是不可变类型

      • 同一个字典中，键（key）必须是唯一的

      • 创建字典

  • 不可变数据：number、string、tuple

  • 可变数据：list、set、dictionary

  • type()函数（不会认为子类是一种父类类型）可查询变量所指的对象类型，也可用isinstance()判断（会认为子类是一种父类类型）

  • 可以用del语句删除一些对象引用

    del var1[,var2[,var3[...,varN]]]

  • 特别的数值运算

    • /：除法，得到一个浮点数
    • //：除法，得到一个整数
    • **：乘方

  • 一个变量可以通过赋值指向不同类型的对象

  • 混合计算时，python会把整型转换为浮点型

数据类型转换

  • 一般只需将数据类型作为函数名
  • 两种数据类型转换
    • 隐式数据类型转换：自动完成
      • 在进行数据运算时会发生转换，新变量的类型转换，参与运算的变量类型不变，若无新变量存储结果，则需要进行显式数据类型转换
    • 显式数据类型转换：需要使用类型函数转换

推导式

  • 用于从一个数据序列构建另一个数据序列

  • 列表推导式

    • ```python
      [表达式 for 变量 in 列表]
      [out_exp_res for out_exp in input_list]

      或者

      [表达式 for 变量 in 列表 if 条件]
      [out_exp_res for out_exp in input_list if condition]
      ‘’’
      out_exp_res:列表生成元素表达式，可以是有返回值的函数
      for out_exp in input_list:迭代input_list将out_exp传入out_exp_res表达式中
      if condition:条件语句，过滤列表中不符合条件的值
      ‘’’

  • 集合推导式

    • {expression for item in sequence}

或

{expression for item in sequence if condition}

    • 元组推导式与列表推导式的用法完全相同

    • 元组推导式返回的结果是一个生成器对象

    • 使用tuple()函数可以直接将生成器对象转换成元组

迭代器与生成器

  • 迭代器
    • 迭代器是一个可以记住遍历的位置的对象
    • 迭代器对象从集合的第一个元素开始访问，直到所有元素被访问完为止，只往前不后退
    • 迭代器两个基本方法：iter()和next()
    • 字符串、列表和元组对象都可以用于创建迭代器
    • 迭代器对象可以用常规for语句遍历
  • 创建一个迭代器
    • 把一个类作为迭代器使用需要在类中实现__iter__()和__next__()方法
    • __iter__()方法返回一个特殊的迭代器对象，这个迭代器对象实现了__next__()方法并通过StopIteration异常标识迭代的完成
    • __next__()方法会返回下一个迭代器对象
  • StopIteration异常用于标识迭代的完成，避免无限循环raise StopIteration
  • 生成器
    • 使用了yield的函数为生成器
    • 生成器是一个返回迭代器的函数，只能用于迭代操作
    • 调用生成器运行的过程中，每次遇到yield时函数会暂停并保存当前所有的运行信息，返回yield的值，并在下一次执行next()方法时从当前位置继续运行

函数

  • 定义函数
    • 函数代码块以def开始，后接函数标识符名称和()
    • 传入参数和自变量必须放在圆括号中间
    • 函数内容的第一行语句可以选择性的使用文档字符串
    • 函数内容以冒号开始并且缩进
    • return语句结束函数，单独的return相当于返回None
  • 参数传递
    • python中类型属于对象，对象有不同类型的区分，变量没有类型，仅仅是对象的引用
    • 可更改对象和不可更改对象
      • 不可变类型：修改值实际上创建新的对象，其参数传递类似C++的值传递
      • 可变类型：修改值是真的修改值，其参数传递类似C++的引用传递
  • 参数
    • 调用函数时可使用的正式参数类型
      • 必须参数
        • 须以正确的顺序传入函数，调用时数量必须和声明时的一致
      • 关键字参数
        • 在圆括号中显式为定义时的形参赋值，允许顺序不一致
      • 默认参数
        • 在定义函数时在形参列表中为形参赋值，调用时若没有传递参数则使用默认参数
      • 不定长参数
        • 加*的参数会以元组的形式导入，存放所有未命名的变量参数
        • 加**的参数以字典的形式导入
        • *也可以单独出现，*后的参数必须用关键字传入
  • 匿名函数
    • python用lambda创建匿名函数
      • lambda只是一个表达式，函数体比def简单的多
      • lambda函数拥有自己的命名空间，且不能访问自己参数列表之外或全局命名空间里的函数
    • 语法
      • lambda [arg1[,arg2,...]]:expression
  • 强制位置参数
    • 单独的/之前必须使用指定位置参数

模块

  • 模块是一个包含所有定义的函数和变量的文件，后缀为.py，模块可以被别的程序引入，这也是使用python标准库的方法
  • sys.argv是包含命令行参数的列表，sys.path是包含python解释器自动查找所需模块的路径的列表
  • 使用模块需要执行import语句
  • 解释器遇到import语句时，如果模块在当前的搜索路径就会被导入
  • 一个模块只会被导入一次
  • 若要导入的模块不在搜索路径中，可以修改sys.path
  • 若经常使用模块内的函数，可以将其赋值给一个本地名称
  • from…import语句，从模块中导入一个指定的部分到当前的命名空间中
  • 模块除了方法 定义，还可以包括可执行的代码，这些代码一般用来初始化这个模块，这些代码只有在第一次被导入时才会被执行
  • 每个模块有各自独立的符号表，在模块内部为所有函数当作全局符号使用
  • 模块可以导入其他模块，被导入的模块名称将被放入当前操作的符号表中
  • from module import *将把所有名字导入，但那些由单一下划线开头的名字不在此例，但引入的其他来源的命名可能覆盖已有的定义
  • __name__属性
    • 一个模块被另一个程序引入时，其主程序将运行
    • 每个模块都有一个__name__属性，当其值是__main__时，表明该模块自身在运行，否则是被引入
  • dir()函数可以找到模块内定义的所有名称，以一个字符串列表的形式返回
  • 标准模块
    • 有些模块直接被构建在解析器里
    • sys模块内置在每一个python解析器中，sys.ps1和sys.ps2定义了主提示符和副提示符所对应的字符串
  • 包
    • 层次在模块之上
    • 导入一个包时，python会根据sys.path中的目录来寻找这个包中包含的子目录
    • 目录只有包含一个叫做__init__.py的文件才会被认作是一个包，这个文件可以包含一些初始化代码或者为__all__变量赋值
    • import语句使用后，调用指定模块中的函数需要带包和模块的名称，最后一个.后可以是包或者模块，之前的必须是包
    • from import语句使用后，被import的包或者模块或者函数会加入当前的命名空间
    • import语句会首先把后面的当作一个包定义的名称，若没找到，再试图按照一个模块去导入，若还没找到，抛出一个:exc:ImportError
  • 从一个包导入*
    • from package.subpackage import *时，python会进入文件系统，找到这个包里面所有的子模块，然后全部导入
    • 由于Windows不区分大小写，这个方法工作的不好，为解决这个问题，需要提供一个精确包的索引
    • 导入语句遵循以下规则：
      • 若包定义文件__init__.py存在一个__all__的列表变量，那么在使用from package import *时就把这个列表中的所有名字作为包内容引入
      • 若__all__没有定义，那么使用时不会导入包中的如何子模块，只是把包和它里面定义的所有内容导入进来（可能运行__init__.py里定义的初始化代码），会把__init__.py里面定义的所有名字导入进来，并且不会破坏掉我们在这句话之前导入的所有明确指定的模块
    • 如果在结构中包是一个子包，想要导入兄弟包，需要使用绝对路径导入
    • 隐式和显式相对导入都是从当前模块开始
    • 主模块名字永远是__main__，python应用程序的主模块应当总是使用绝对路径引用
    • 包提供属性__path__，这是一个目录列表，里面每一个包含的目录都有为这个包服务的__init__.py，需要在其他__init__.py执行前定义，修改这个变量可以影响包含在包里面的模块和子包

      • str()：函数返回一个用户已读的表达形式
      • repr()：产生一个解释器易读的表达形式
    • rjust()：将字符串靠右并在左边填充空格
    • ljust()和center()类似，zfill会在左边填充0

      • 格式化字段中可使用!a（ascii()），!s（str()），!r（repr()）在格式化某个值之前对其进行转化

  • 读和写文件

    • open函数将会返回一个file对象
    • 基本语法：open(filename,mode)
    • 文件对象的方法
      • read方法：可传入size参数，读取一定数目的数据，然后作为字符串或字节对象返回
      • readline方法：从文件读取单独的一行，若返回一个空字符串说明到达最后一行
      • readlines方法：返回该文件中包含的所有行，若设置可选参数sizehint，则读取指定长度的字节，且按换行符分隔
      • write方法：将参数string写入到文件中，然后返回写入的字符数
      • tell方法：返回文件对象当前所处的位置，是从文件开头开始算起的字节数
      • seek方法：改变文件指针当前的位置，f.seek(offset,from_what)，from_what是0则表示开头，1则表示当前位置，2则表示结尾，默认0
      • close方法：关闭文件并释放系统资源
    • 处理一个文件对象时，使用with关键字比较好

    • 反序列化操作可以从文件中创建上一次程序保存的对象
    • 基本接口：pickle.dump(obj,file[,protocol])
    • 有了pickle这个对象就能对file以读取的形式打开，x=pickle.load(file)，从file中读取一个字符串并将他重构为原来的python对象
